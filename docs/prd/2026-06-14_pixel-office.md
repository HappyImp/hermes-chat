# 像素风办公室 PRD

**日期**: 2026-06-14
**状态**: ✅ 已完成

## 1. 功能描述

在主内容区域右侧/下方增加一个像素风办公室可视化组件，用 8-bit 风格展示员工的工位和在线状态。每个员工有一个固定的像素工位，在线时显示像素角色动画，离线时工位为空。鼠标悬停显示气泡对话框，展示员工当前任务。

## 2. 用户故事

- 作为用户，我希望看到一个有趣的像素风办公室，让工作界面不那么枯燥
- 作为用户，我希望在线员工的工位有像素角色，离线时工位为空，直观反映状态
- 作为用户，我希望鼠标悬停在角色上时显示气泡，显示该员工当前在做什么
- 作为用户，我希望办公室布局是固定的，每个员工有固定的"座位"，方便快速定位

## 3. 验收标准

- [x] 像素风办公室组件（Canvas 或 SVG 渲染）
- [x] 5 个员工工位：固定位置，网格布局
- [x] 在线员工显示像素角色（idle 动画）
- [x] 离线工位显示空桌椅
- [x] 鼠标悬停显示 SpeechBubble（任务描述 + 时长）
- [x] 状态数据来自 useEmployeeStatus hook（复用）
- [x] 响应式：移动端纵向排列，桌面端横向排列

## 4. 技术约束

- **渲染方式**: HTML/CSS 像素风格（Canvas 可选）
- **精灵图**: 内嵌 TypeScript sprite 数据（sprites.ts）
- **布局**: 固定网格，5 个工位
- **动画**: CSS animation（idle 呼吸动画）
- **气泡**: 绝对定位 + CSS 箭头
- **数据源**: useEmployeeStatus hook（与侧边栏共享）

## 5. 优先级

| 功能 | 优先级 | 原因 |
|------|--------|------|
| 工位渲染 | **P1** | 核心可视化 |
| 状态联动 | **P1** | 数据驱动而非静态展示 |
| 气泡对话框 | **P2** | 交互增强 |
| 动画效果 | **P2** | 体验优化 |
