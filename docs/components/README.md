# 组件文档

## 1. Sidebar

侧边栏容器，包含 Channel 列表、会话列表和员工状态面板。

## 2. ChannelList

Channel 管理组件。

## 3. SessionList

会话列表组件。

## 4. EmployeeStatus

员工实时状态面板。

## 5. ChatArea

聊天区域容器。支持 `/dispatch` 命令触发员工任务。

## 6. MessageBubble

单条消息气泡组件。支持 TaskCard 渲染。

## 7. MessageInput

消息输入框组件。

## 8. Welcome

空会话欢迎页面。

## 9. CodeBlock

代码块组件，支持复制功能。

## 10. Toast

全局提示组件。

## 11. TaskCard

员工异步任务卡片组件，显示任务状态、进度和结果。
