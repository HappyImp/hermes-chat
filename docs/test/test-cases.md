# Hermes Chat 测试用例文档

**创建日期**: 2026-06-14
**版本**: v1.0
**状态**: 活文档，持续维护
**测试环境**: 
- 线上: http://43.249.192.131:7960/chat/
- 本地: http://localhost:5244/chat/

## 测试概览

### 功能模块
1. **核心聊天功能** (F1, F4)
2. **会话管理** (F2)
3. **Channel 管理** (F3)
4. **导出对话** (F5)
5. **员工状态面板**
6. **像素办公室**
7. **响应式布局**
8. **API 代理**

### 测试类型
- 功能测试：验证功能是否符合 PRD 要求
- 交互测试：用户操作流程和体验
- 边界测试：异常情况和边界条件
- 响应式测试：不同设备尺寸的适配
